Reporting Vulnerabilities¶

Send a mail to Uri Arev <me@wantyapps.xyz> with the following syntax:

Subject: [VULN] <explanation/summary of the vulnerability>

Body: How to exploit/reproduce bug, what lines/file it is in and (optional) how to fix it.

You can also send a patch file or a diff.
